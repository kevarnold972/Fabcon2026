## About

Carlos Robles is a technology leader and Principal Product Manager at Microsoft, shaping the future of SQL development experiences across the Microsoft ecosystem. He leads the vision and delivery of the MSSQL extension for VS Code and its deep integration with GitHub Copilot, empowering developers to build intelligent, data-driven applications faster and more intuitively.

With over 16 years of experience spanning software engineering, solutions architecture, data architecture, and product strategy, Carlos combines technical depth with customer empathy to drive adoption and innovation. His expertise includes JavaScript, TypeScript, Python, and SQL across multiple database systems, along with modern DevOps and Infrastructure as Code practices.

An international speaker, author, and mentor, Carlos is passionate about advancing developer productivity and bridging the gap between AI and applications using SQL Server as backend databases.
