## About

Guy leads the Azure Database for PostgreSQL Flexible Server product team. Previously, he was a tech strategy lead for Azure App Innovation in Microsoft's Customer Success Unit (CSU), and various program manager roles including SQL Server cloud infrastructure, Azure Core Compute and the Azure Linux team. In his spare time, he likes to write code and explore the Pacific Northwest.
