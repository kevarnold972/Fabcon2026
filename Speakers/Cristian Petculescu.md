## About

I am a Technical Fellow at Microsoft, and, in particular, the Chief Architect of Azure Data.

Azure Data includes:

1. Microsoft Fabric
2. The Intelligence Platform: Power BI, Azure Analysis Services, Azure Synapse (DataWarehouse, Spark, Azure Data Factory, Azure Data Explorer aka Kusto), Cosmos, Azure Messaging.
3. The operational databases group: SQL Server, Azure SQL DB, Cosmos DB, Azure MySQL, Azure PostgreSQL.
4. Azure Databricks

I have been working inside Microsoft since 1997, after I have received the MSc degree in computer science from 'Politehnica' University of Bucharest. My responsibilities include technical strategy for the products of the Azure Data, technical tactical guidance for the teams of the division, and ... well ... a lot of coding. I am an author of 49 granted patents, and many more pending. I have received my PhD degree in Computer Science from 'Politehnica' University of Bucharest in 2006.

I am one of the core co-inventors of the SQL Server in-memory technology (Vertipaq) - the technology that is the backbone of the columnar indexes, as well as of the tabular storage engine of Analysis Services
