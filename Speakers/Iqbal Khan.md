## About

Iqbal Khan is a Product Engineer at Microsoft, helping develop the latest Power BI and Fabric features to assist organizations in their deployment of analytics-based technologies.

In his role, Iqbal serves as the technical lead for the Admin monitoring workspace feature set provided in Power BI/Fabric. The Admin monitoring workspace is designed to provide admins with cutting-edge capabilities to conduct security and governance-related tasks such as audits and usage checks.
