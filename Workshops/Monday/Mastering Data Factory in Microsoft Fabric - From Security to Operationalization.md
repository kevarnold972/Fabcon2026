## Description

Fabric Data Factory enables organizations unify their data estate with OneLake. Security is paramount.

You will learn how to secure your data ingestion, transformation using workspace identity auth and Azure Key Vault integration, implement network isolation (Private Link and VNet gateway) for accessing data behind firewall.

Operationalization is another important aspect. You will learn to operationalize pipeline, dataflow gen2 with variable library, build metadata driven pipelines and more.
